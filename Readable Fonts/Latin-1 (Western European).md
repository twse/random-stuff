## Fonts
The WST fonts discontinued with Windows 7 are designed for use with terminal programs. The encoding is MS Symbol but the fonts work fine with word wrapping. The range 128-255 includes a set of Braille characters.

### Characters for fonts used in games by KidStuff Software
Pay shareware games by KidStuff Software (e.g. Preschool Pack) includes a font used for the settings window and uses their own encoding based on CP437. Box Drawing and other symbols are empty.

| Glyph     | Codes   |
|-----------|---------|
| sharp s   | Alt-170 |
| o slash   | Alt-176 |
| O slash   | Alt-177 |
| copyright | Alt-178 |

When entering non-ASCII characters in the address in Preschool Pack, the mailing box font in the last activity shows up as blank.
